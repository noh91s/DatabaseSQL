 연산자

연산자 우선순위
- 곱하기와 나누기는 더하기와 빼기보다 우선순위가 높습니다
- 같은 우선순위의 연산자는 좌측에서 우측으로 계산됩니다
- 괄호는 강제로 계산의 우선순위를 바꾸거나 문장을 명료하게 하기 위해 사용됩니다

1. 산술 연산자
+ - *  / 덧셈, 뺄쎔, 곱셈, 나눗셈

2. 비교 연산자
= "같다" -> 일치하는 것만
> < >= <= "비교"
<> != ^= "같지 않다"

3. 논리 연산자
and	조건1 and 조건2 -> 두조건 모든 true 일 때 조회
or	조건1 or 조건2 -> 하나 조건 true 일 때 조회
not 	not 조건 -> 조건이 true가 아닐 때 조회

4. 기타
between ~사이 between 1 and 10 - 1~10
in	포함 in(1,2,3) 1,2,3 포함되어 있는
like	칼럼 like '%' -> 포함되어 있는 모든 조회(검색)-> "~포함하는" contian
	칼럼 like 'a%' -> a로 시작하는 모든
	칼럼 like '%a%' -> a가 포함되어 있는 모든

문자 연산자 || 이 있는데, 두 문자를 연결하는 연산을 수행한다 
사실 , where문에 쓰이기보다는
select ~ from ~ where 에서 select 쪽에 많이 쓰인다.


별칭 as , 생략 -> join 테이블이 여러개 일대 많이 사용
테이블 as 별칭;
칼러명 as 별칭;

오라클 정렬 > 오름차순(기본), 내림차순 -> 조회
오라클에서 검색된 결과의 행을 정렬할 때는 ORDER BY 절을 사용
1. 오름차순 정렬(ascening sort) -> 작은 값 부터 큰 값으로 정렬 -> 기본 (생략하면) ASC
 	1, 2, 3, 4,,,,,, 강 나 도 류 민 박 성 양 조 차 ㅋ 탁 표 황
2. 내림차순 정렬(descending sort) 정렬 -> 큰 값 부터 작은 값으로 정렬 -> DESC
	100, 99, 98,,,,,
