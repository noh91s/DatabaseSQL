 단축키 모음 ( SQL )

CTRL + S : SQL 워크시트 파일 저장

CTRL + F : 검색

CTRL + C : 선택된 행 복사

CTRL + F : 검색(특정 단어 클릭후 해당 단축키 누르면 모두 검색됨)

CTRL + SHIFT + D : 위의 한행 복사 붙여 넣기

ALT + SHIFT + D : 한 줄 삭제

CTRL + F7 : 선택된 영역 정렬 

CTRL + R : 테이블 새로고침(테이블 선택하고)

TAB : 내어쓰기

SHIFT + TAB : 들어 쓰기


CTRAL + A : 전체선택


ALT + F10  : 데이터베이스 접속 선택창 생성

 

CTRL + SHIFT + D (스크립트 출력 클릭후) : 스크립트 출력창 지우기
